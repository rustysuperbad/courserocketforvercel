# Course-Rocket
CourseRocket.ai is a web application that makes custom courses completely designed based on what the user is working on, or wants to learn.
Main functionalities of the application -

<br> 1> accquiring youtube videos in the order of relevance of learning. - youtube api
<br> 2> research papers sourced from throughout the internet. - papers api
<br> 3> ai- generated course content that helps in learning. - groq api

<br> authentication powered by firebase.
